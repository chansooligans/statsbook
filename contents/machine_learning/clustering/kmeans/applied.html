With clustering problems, you don’t have a labelled training set. But for this demo, we generate y for fake data to be able to evaluate results.

centers = [(-3, -3), (4, 4), (4, -4)]
cluster_std = [2, 3, 2]
X, y = make_blobs(
    n_samples=500, 
    cluster_std = cluster_std, 
    centers = centers, 
    n_features = 2, 
    random_state=0
)

plt.scatter(X[:,0], X[:,1], c=y, cmap="Set1")
